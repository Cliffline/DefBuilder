# DefBuilder

![Last Updated](https://img.shields.io/github/last-commit/Cliffline/DefBuilder?path=README.md&display_timestamp=committer&label=last%20updated&style=flat-square)

DefBuilder is a Steam Inventory item definition editor.

The repository contains the reference documentation that defines the product scope and the Steamworks rules the editor needs to model. It also contains a minimal web prototype for importing, editing, validating, and exporting ItemDef JSON.

- Documentation home: [docs/README.md](docs/README.md)
- Steam Inventory reference: [docs/reference/steam-inventory/README.md](docs/reference/steam-inventory/README.md)

## Web Prototype

- Stack: React, TypeScript, Vite, and Zod
- Current focus: local JSON import, common field editing, validation feedback, and JSON export
