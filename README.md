# DefBuilder

DefBuilder is a browser editor for Steam Inventory ItemDef JSON. [![Open on itch.io](https://img.shields.io/badge/open%20on-itch.io-FA5C5C?logo=itchdotio&logoColor=white)](https://clifflinegames.itch.io/defbuilder)

Import, edit, validate, and export Steam Inventory item definitions in the browser.

## Web App

- Stack: React, TypeScript, Vite, and Zod
- Current focus: local JSON import, common field editing, validation feedback, and JSON export
